# chinese2japanese

複数の中国語の単語をまとめて日本語に変換します。
出力はマークダウン形式です。

## 必要なもの

- Python 3.x

## インストール

```
pip install -r requirements.txt
```

## 実行方法

chinese2japanese のフォルダ内に単語の間を"，"で区切った羅列をファイルを入れます。
その後 src/app.py を実行し指示の通りにファイル名を入力します。
